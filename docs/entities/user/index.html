    User ¶

    Users in Microsoft Dynamics 365 are individuals who have specific logins and passwords and a set of attached privileges at various access levels.

    Adding Users to CRM¶

    To add a User to CRM:

    1. In your Azure Active Directory Admin Centre, select the User you wish to add to CRM.

    2. In 'Licenses & Apps', assign a 'Microsoft Flow Free' license.

    3. Then go to 'Groups'. Select the 'Dynamics' group. Add the User to the group. Click 'Save/Apply'.

    4. Within 1-10 minutes the =User will appear in CRM. The new User will have the most basic Security Role which they inherit from the default Team within CRM.

    5. Open the User in CRM. Expand the 'Administrator' tab in the User form. Click the 'Mailbox' to open it.

    6. Click 'Approve Email' and then 'Test & Enable Mailbox'. Refresh the page until 3 successes.

    Further: How to install 'Dynamics 365 App for Outlook' addin.

    Updating your Mobile Phone number¶

    1. Go to www.office.com

    2. Click your User 'bubble' (top-right)

    3. Click 'My Office Profile'

    4. Click 'Update profile'
    5. Edit your mobile number

    Other¶

    • It is not possible to add a 'Social Pane' (Activity Wall) to the 'User' form.
